# 3-Tier Web Application (PHP + MySQL + Frontend)

## Overview
This is a simple 3-tier web application demonstrating:
- Frontend (HTML + CSS)
- Backend (PHP APIs)
- Database (MySQL)

Users can add and view messages stored in the database.

---

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP (PDO)
- Database: MySQL
- Containerization: Docker (docker-compose)

---

## Project Structure
frontend/ → UI files  
backend/ → PHP APIs  
database/ → SQL schema & initial data  
docker-compose.yml → Container setup  

---

## Features
- Add new messages
- View latest messages
- REST-style PHP API
- Dockerized 3-tier architecture
